# modules-finance

The **Finance Module** is a core component of the Dorcas Hub that provides finance &amp; accounting related functionality to their business.

## features
- Manage Accounts (such as Journals and/or transaction categories)
- Manage custom Sub-Accounts
- Manage Entries
- TransTrak (automatically import email transaction alerts)

## using the module
To get started, you can either [sign-up](https://dorcas.io/business/cloud) for the **cloud** version or [download](https://dorcas.io/download/) and install the **self-hosted** version

> Step 1: Login to your Hub account
> Step 2: Click on the **Sales** menu
